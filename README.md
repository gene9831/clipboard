# 超简单剪切板

`flask` + `vue.js` + `element-ui.js` 实现的超简单局域网剪切板，数据存储在内存中。

## 环境

### 前端

- `vue 2.6.11`
- `vue-clipboard2 0.3.1`
- `element-ui 2.13.2`
- `axios 0.19.2`

### 后端

- `python 3.8`
- `Flask-RESTful 0.3.8`
- `waitress 1.4.4`

## 安装依赖

```shell
pip3 install -r requirements.txt
```

## 使用

```shell
python3 app.py
```
